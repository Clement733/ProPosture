## Getting Started
### Setup

Navigate to the base level of the repository
```
cd {your/path/here}/Proposture
```

Install requirements
```
pip install -U pip
pip install -r requirements.txt
```

### Local Streamlit
```
streamlit run MLOps/streamlit_app.py
```
